
1. Clone the repository:

```bash
   git clone https://github.com/Sirithonk2002/frontend-task.git
   cd frontend
```
2. Install dependencies
```bash
    npm install
 ```
3. Install React Icons
```bash
    npm install react-icons --save  

```
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
